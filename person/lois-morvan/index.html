Loïs Morvan, Développeur TypeScript / React / Next.js

Applications web, MVP et outils métier

Bonjour, je suis développeur freelance spécialisé TypeScript, React, Next.js et Node.js, basé en Pays de la Loire. J'aide les entreprises à concevoir et livrer des applications web, MVP, back-offices, dashboards et outils métier clairs, rapides et maintenables.

Je privilégie les produits web sur mesure : interfaces React lisibles, parcours simples, logique backend propre, intégrations d'API, authentification et bases de données bien structurées. Mon objectif est de livrer vite, sans transformer les premières itérations en dette technique difficile à reprendre.

Mon expérience vient notamment de trois ans d'alternance chez LNA Santé, sur des projets internes à fort impact : plateforme achats utilisée par environ 80 collaborateurs, restriction de catalogues par profil métier, commandes SAP, statuts automatisés, notifications, SSO Azure AD et application de réservation interne avec front React.

J'ai également réalisé des projets produits comme Nutria, une application Next.js de nutrition avec planification par IA, dashboard personnalisable et listes de courses intelligentes, ainsi que Crèche Studio, un outil métier de gestion, rapprochement facture-enfant, calcul de chiffre d'affaires et export PDF pour photographe indépendante.

Si vous avez besoin de transformer une idée, un process interne ou un outil existant en application web claire et robuste, je serai ravi d'en discuter.
